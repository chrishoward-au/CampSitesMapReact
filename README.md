# Camp Sites Map React

An interactive map application for exploring camp sites, built with:

- React
- TypeScript
- Vite
- Tailwind CSS
- Preline UI
- Mapbox GL JS
- Cloudflare Pages

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Mapbox account and access token

### Installation

1. Clone the repository
2. Install dependencies:

   npm install
